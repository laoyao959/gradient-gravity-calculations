# Gradient Material Field and Geodesic Network

**Calculation codes for the paper:**

Gradient Material Field and Geodesic Network: Atomic Ontology of Gravity and Geometric Equivalent Interpretation of General Relativity

### Files in this repository:
- `Mercury_perihelion_precession.ipynb` — 水星近日点进动计算
- `GW230529_time_difference.ipynb` — GW230529引力波时间差计算
- Python scripts for vector superposition and propagation speed

All codes are open-source and fully reproducible.

GitHub: https://github.com/laoyao959/gradient-gravity-calculations
